# Real-Time Vehicle Monitoring System (ESP32)

## Overview
Developed an embedded system using ESP32 to monitor vehicle conditions in real time using sensor data.

## Features
- GPS-based location tracking
- Accelerometer-based tilt and motion detection
- Real-time monitoring and visualization

## Connectivity
- Wi-Fi for real-time data transmission
- GPS module for location tracking

## Working Principle
Sensor data from accelerometer and GPS is collected using ESP32 and processed to detect abnormal conditions such as sudden tilt or speed variation. The data is transmitted via Wi-Fi and displayed on a real-time dashboard.

## Technologies Used
- ESP32
- Embedded C / Arduino IDE
- Sensors (Accelerometer, GPS)

## Folder Structure
- code/ → ESP32 source code

## Output
- Real-time vehicle tracking
- Detection of abnormal events
- Live dashboard visualization
